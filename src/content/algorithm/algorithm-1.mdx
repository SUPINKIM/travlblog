#### [문제]
>마인은 곡괭이로 광산에서 광석을 캐려고 합니다. 마인은 다이아몬드 곡괭이, 철 곡괭이, 돌 곡괭이를 각각 0개에서 5개까지 가지고 있으며, 곡괭이로 광물을 캘 때는 피로도가 소모됩니다. 각 곡괭이로 광물을 캘 때의 피로도는 아래 표와 같습니다.
![](https://velog.velcdn.com/images/qorgkr26/post/265187d2-8e81-4ed2-ae8e-b52777affa72/image.png)
예를 들어, 철 곡괭이는 다이아몬드를 캘 때 피로도 5가 소모되며, 철과 돌을 캘때는 피로도가 1씩 소모됩니다. 각 곡괭이는 종류에 상관없이 광물 5개를 캔 후에는 더 이상 사용할 수 없습니다.
마인은 다음과 같은 규칙을 지키면서 최소한의 피로도로 광물을 캐려고 합니다.
사용할 수 있는 곡괭이중 아무거나 하나를 선택해 광물을 캡니다.
한 번 사용하기 시작한 곡괭이는 사용할 수 없을 때까지 사용합니다.
광물은 주어진 순서대로만 캘 수 있습니다.
광산에 있는 모든 광물을 캐거나, 더 사용할 곡괭이가 없을 때까지 광물을 캡니다.
즉, 곡괭이를 하나 선택해서 광물 5개를 연속으로 캐고, 다음 곡괭이를 선택해서 광물 5개를 연속으로 캐는 과정을 반복하며, 더 사용할 곡괭이가 없거나 광산에 있는 모든 광물을 캘 때까지 과정을 반복하면 됩니다.
마인이 갖고 있는 곡괭이의 개수를 나타내는 정수 배열 picks와 광물들의 순서를 나타내는 문자열 배열 minerals가 매개변수로 주어질 때, 마인이 작업을 끝내기까지 필요한 최소한의 피로도를 return 하는 solution 함수를 완성해주세요.


#### [핵심 아이디어]
일단 보자마자 연속된 5개 가운데 다이아몬드가 제일 많은 row순으로 다이아 > 철 > 돌 곡괭이를 쓰면 되겠다고 아이디어는 떠오르는데 어떻게 풀어야 될 지 감이 안 와서 고민하다가 다른 사람들 질문 속에 힌트를 얻고 풀었다. 

풀이 방법은 다양한 거 같은데 처음에 우선순위 큐로 풀려다가 어차피 광물의 개수가 50개가 최대라서 sorting으로도 충분히 풀 수 있을 것 같아서(n^2 라고 해봤자 2500개) 5개씩 광물을 확인해 2차원 배열로 만들어주었다. 하나의 row가 한 번에 캘 수 있는 미네랄이고 각 컬럼은 다이아, 철, 돌 순으로 만들어주었다.  그 후 다이아 > 철 > 돌 순으로 광물 배열을 sorting 해주었다. 

여기서 하나 놓쳤던 점이 [광물은 주어진 순서대로만 캘 수 있다]인데 만약 전체 사용 가능한 곡괭이 수 * 5보다 광물 개수가 더 크면 sorting 할 때 캘 수 없는 광물까지 캘 수 있는 것으로 카운팅 해서 오히려  피로도가 더 높게 나올 수 있다. 그래서 minerals 개수와 곡괭이 * 5 를 비교해서 slice를 먼저 해주는 코드가 꼭 필요하다. (안 그럼 8번 테케에서 실패난다.)

#### [정답 코드]
```js
function solution(picks, minerals) {
    const pick_map = {
        dia : picks[0],
        iron : picks[1],
        stone : picks[2],
    };
    
    const row_count = Math.floor(minerals.length / 5) + 1;
    
    const minerals_map = Array.from(new Array(row_count), () => new Array(3).fill(0)) // dia, iron, stone
    
    const picks_count = picks.reduce((acc,cur) => acc += cur);
    
    if (picks_count * 5 < minerals.length) {
        minerals = minerals.slice(0, picks_count * 5);
    }
    
    for (let i = 0; i < minerals.length; i++) {
        const x = Math.floor(i / 5);
        const now_mineral = minerals[i];
        if (now_mineral === 'diamond') minerals_map[x][0] += 1;
        if (now_mineral === 'iron') minerals_map[x][1] += 1;
        if (now_mineral === 'stone') minerals_map[x][2] += 1;
    }

    minerals_map.sort((a,b) => {
        if (b[0] === a[0]) {
            if (b[1] === a[1]) {
                return b[2] - a[2];
            }
            return b[1] - a[1];
        }
        return b[0] - a[0];
    });
    
    let answer = 0;
    while (minerals_map.length) {
        const [d, i, s] = minerals_map.shift();
        
        const pick_using = Object.entries(pick_map).find(pick => pick[1]);
        
        if (!pick_using) break;
        
        if (pick_using[0] === 'dia') {
            answer += (d + i + s);
        } else if (pick_using[0] === 'iron') {
            answer += (d * 5 + i + s);
        } else {
            answer += (d * 25 + i * 5 + s);
        }
        pick_map[pick_using[0]]--;
    }
    
    return answer;
}
```
